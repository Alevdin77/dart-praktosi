Автосалон – CLI система на Dart

Предметная область
Автосалон. Программа хранит информацию об автомобилях, покупателях и оформленных продажах.

/ Структура папок
bin/main.dart - главный файл для запуска
lib/src/domain/models - классы Car, Customer, Sale
lib/src/domain/validators - функции для проверки данных
lib/src/data/database.dart - подключение к SQLite и создание таблиц
lib/src/data/repositories - добавление, просмотр, изменение, удаление
lib/src/cli - меню и функции для ввода данных

/ Слои модульности
domain - только модели и правила проверки. Не знает о базе данных и консоли.
data - работа с SQLite и репозитории. Использует domain, но не знает о cli.
cli - меню и ввод данных. Использует и domain, и data.

/ Валидации
1. Текстовое поле не может быть пустым (проверка через trim)
2. Цена автомобиля должна быть больше 0
3. Год выпуска от 1900 до текущего года (сейчас 2026)
